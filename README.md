## Spog UI

### Environment variables

If needed, you can configure the ENV VARS. E.g.

```shell
export [VAR_NAME]=[VAR_VALUE]
```

| Description                   | ENV VAR       | Defaul value          |
| ----------------------------- | ------------- | --------------------- |
| Set Guac API URL              | GUAC_HUB_URL  | http://localhost:8083 |
